## Summary
`Math.random()` is used to generate OTPs in `src/app/api/auth/send-otp/route.ts` (line 7). `Math.random()` is not a cryptographically secure pseudo-random number generator (CSPRNG) and its output can be predicted in some environments.

## Location
`src/app/api/auth/send-otp/route.ts`, line 6–8:
```ts
function generateOTP(): string {
  return Math.floor(100000 + Math.random() * 900000).toString();
}
```

## Fix
Replace with Node.js `crypto.randomInt`:
```ts
import crypto from "crypto";

function generateOTP(): string {
  return crypto.randomInt(100000, 1000000).toString();
}
```

## Severity
Security — Medium
